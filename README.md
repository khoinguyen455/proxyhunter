## Proxy Hunter V1.0.0 - What's New/Có gì mới?
### English
- Supports adding and removing sources
- Allows selecting sources with checkboxes
- Supports multiple proxy types:
  - HTTP
  - HTTPS
  - SOCKS4
  - SOCKS5
- Checks proxies with highly accurate results
- Displays essential information:
  - Anonymity Level
  - ISP
  - Network Type
- Can detect whether a proxy is:
  - Datacenter
  - Residential

- Supports re-checking proxies with `UNKNOWN` ISP or Network
- `Only fix UNKNOWN ISP/network` is enabled by default
- When enabled:
  - Only proxies with `UNKNOWN` ISP or Network will be re-checked
- When disabled:
  - All `UNKNOWN` proxies will be checked again automatically
- After re-checking:
  - If the ISP or Network still remains `UNKNOWN`
  - The proxy will automatically be removed from the file

- Fully customizable settings and interface:
  - Language selection
  - Light/Dark theme
  - CUSTOM PALETTE for changing UI colors
- Supports adding or removing JUDGES
- Includes JUDGES ping checking support

### Tiếng Việt

- Hỗ trợ thêm và xóa source
- Cho phép tick chọn source cần sử dụng
- Hỗ trợ nhiều loại proxy:
  - HTTP
  - HTTPS
  - SOCKS4
  - SOCKS5

- Kiểm tra proxy với độ chính xác cao
- Hiển thị đầy đủ thông tin cần thiết:
  - Mức ẩn danh
  - ISP
  - Loại mạng
- Có thể xác định proxy thuộc:
  - Datacenter
  - Residential

- Hỗ trợ kiểm tra lại proxy có ISP hoặc Network là `UNKNOWN`
- Tùy chọn `Only fix UNKNOWN ISP/network` được bật mặc định
- Khi bật:
  - Chỉ các proxy có ISP hoặc Network là `UNKNOWN` mới được kiểm tra lại
- Khi tắt:
  - Toàn bộ proxy `UNKNOWN` sẽ được kiểm tra lại tự động
- Sau khi kiểm tra lại:
  - Nếu ISP hoặc Network vẫn là `UNKNOWN`
  - Proxy đó sẽ tự động bị xóa khỏi file

- Giao diện và cài đặt có thể tùy chỉnh:
  - Chọn ngôn ngữ
  - Theme Sáng/Tối
  - CUSTOM PALETTE để đổi màu giao diện
- Hỗ trợ thêm hoặc xóa JUDGES
- Có hỗ trợ kiểm tra ping của JUDGES
